# Raymi Fotografía

Sitio web profesional de **Raymi Fotografía**, servicio fotográfico en Lima, Perú. Actualmente en línea y funcionando.

## Secciones de la web

- **Hero** — Presentación con eslogan y llamado a la acción.
- **Galería por categorías** — Filtro interactivo para ver el portafolio por tipo: bautizos, paisajes, urbanos, maternales, motos, dentales, pedidas de mano.
- **Mosaico** — Photowall tipo collage con fotos distribuidas de forma dinámica.
- **Servicios** — Lista de todos los servicios ofrecidos con descripción breve.
- **Packs y precios** — Tarjetas con planes, precios en soles y características.
- **Sobre mí** — Breve presentación del fotógrafo con estadísticas.
- **Contacto** — Formulario funcional que envía mensajes directamente al correo del cliente.

## Funcionalidades

- Navegación suave con scroll interno.
- Modal para ver fotos en grande.
- Cursor personalizado con efecto visual.
- Formulario de contacto funcional (envío de correos).
- Diseño responsivo adaptado a celulares, tablets y escritorio.
- Animaciones sutiles al hacer scroll.

## Diseño e interfaz

Paleta de colores cálida y natural con tonos terracota, marrón y crema. Tipografía elegante y minimalista. Estilo visual enfocado en dejar que las fotografías sean las protagonistas.

## Tecnologías aplicadas

React, Vite, CSS Modules, EmailJS para el formulario, y Supabase como backend ligero.
